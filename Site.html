O que é um Controlador Lógico Programável (CLP)?

O CLP é o cérebro da automação industrial e residencial, substituindo relés e temporizadores por lógica programável.

Sua arquitetura básica consiste em:

  • CPU: Processa a lógica de programa
  • Memória: Armazena programa e dados
  • Entradas: Recebem sinais de sensores e botões
  • Saídas: Controlam atuadores e dispositivos
  • Fonte: Alimenta todo o sistema

Ciclo de Varredura

1
Leitura das entradas
2
Execução da lógica programada
3
Atualização das saídas

Na Prática: Controlando uma Bomba com o WEG CLIC02

Objetivo

Controlar uma bomba de piscina com sistema de selo (auto-retentivo) usando:

  • I1 (Liga): Botão normalmente aberto (NA)
  • I2 (Desliga): Botão normalmente fechado (NF)
  • Q1: Saída para contator da bomba

Lógica de Selo

O selo mantém o circuito energizado mesmo após soltar o botão de ligar, até que o botão de desligar seja pressionado.

Exemplo desenvolvido no simulador WEG WLP.

Diagrama Ladder

|—[ I1 ]—+—|/|—( Q1 )—|
| | I2
|—[ Q1 ]—+

1. Quando I1 é pressionado, Q1 energiza

2. Q1 mantém o circuito mesmo após soltar I1

3. I2 (NF) desenergiza Q1 quando pressionado

Bônus: Temporizador de 2 horas

|—[ I1 ]————( T1 )—|
|—[ T1 ]————( Q1 )—|

Adicionando um temporizador para desligamento automático após 2 horas:

  • T1: Temporizador de 2 horas (7200 segundos)
  • Quando I1 é pressionado, T1 inicia contagem
  • Q1 permanece ligado até T1 completar

O Projeto Além dos Fios

No projeto "Além dos Fios", o CLP é responsável por monitorar sensores e controlar automaticamente os circuitos elétricos, ligando e desligando cargas conforme a necessidade. Isso gera eficiência energética e modernização sustentável.

O sistema permite:

  • Controle remoto via interface web
  • Programação de cenários personalizados
  • Monitoramento em tempo real
  • Relatórios de consumo energético
